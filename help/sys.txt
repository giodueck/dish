Permite ejecutar comandos del sistema.
Uso:
    sys [COMANDO, OPCIONES, ARGUMENTOS]
    sys [OPCION]

    COMANDO     Comando a ejecutarse
    OPCIONES    Opciones para el comando a ejecutarse
    ARGUMENTOS  Argumentos para el comando a ejecutarse
     Ojo: los comandos no propios de este shell no se
          documentan desde DISH.

    OPCIONES
     -h,    --ayuda     Muestra esta ayuda
